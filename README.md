# Big Blue Button Playback-Speed - Chrome-Extension

Change playback speed of bbb recordings.

## Install

* Download the latest release from the [releases tab](https://github.com/juliankeppler/bbb-playbackspeed-chrome-extension/releases/latest)
* Unzip the file
* Load the folder as extension in Google Chrome ([see description here](https://developer.chrome.com/extensions/getstarted#manifest))
